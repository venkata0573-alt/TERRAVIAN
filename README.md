# Terravian Landscaping Business Application

A React-based business application built for **Terravian Landscaping LLC** to manage digital presence, customer inquiries, service information, and lead generation for landscaping and snow management services.

## Overview

Terravian Landscaping Business Application is a responsive front-end application designed to support a local property maintenance business with service presentation, customer communication, lead capture, chatbot assistance, and local SEO visibility.

The application helps customers learn about services, explore landscaping and snow management options, and submit inquiries through a structured contact flow. It is built with a modern React stack and designed for performance, usability, and future business expansion.

## Key Features

* Responsive React application
* Business landing page and service sections
* Landscaping, snow plowing, seasonal cleanup, and hardscaping service flows
* Interactive chatbot for customer guidance
* Contact form with Supabase integration
* Local SEO support with React Helmet
* Sitemap and robots.txt configuration
* Animated UI using Framer Motion
* Mobile-friendly navigation
* Reusable UI components
* Lazy-loaded images for performance
* Trust-focused business sections for local customers

## Business Purpose

This application was built to help Terravian Landscaping LLC create a professional digital system for customer acquisition, service discovery, quote requests, and business credibility.

Instead of only displaying information, the application is structured to support real business outcomes: generating leads, improving customer trust, simplifying service communication, and creating a scalable foundation for future operational tools.

## How to Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
